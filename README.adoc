App Android avanzata per la gestione professionale di allenamenti a intervalli (HIIT, Tabata, Circuit Training).
Progettata per funzionare completamente offline, rispettando la privacy dell'utente.

== 📱 Funzionalità Principali

* **Gestione Allenamenti Flessibile**: Creazione di esercizi con round personalizzati, serie multiple, durate di lavoro e pause variabili.
* **Timer Circolare Visivo**: Interfaccia grafica personalizzata con feedback visivo dei progressi.
* **Feedback Vocale (TTS)**: Text-to-Speech avanzato che annuncia esercizi, round, pause e conteggi alla rovescia.
* **Condivisione Smart**: Condivisione di interi allenamenti tramite Link, senza bisogno di registrazione o server cloud.
* **Risparmio Energetico**: Modalità specifica per schermi AMOLED e riduzione del consumo CPU.
* **Privacy First**: I tuoi dati rimangono sul tuo dispositivo.

== 🔗 Come funziona la Condivisione

L'app utilizza un sistema innovativo per condividere gli allenamenti senza passare dati su internet:

1. L'utente clicca "Condividi" su un allenamento.
2. L'app genera un link sicuro contenente tutti i dati dell'allenamento (criptati nel link stesso).
3. Quando il destinatario apre il link:
   * Se l'app è installata -> Si apre automaticamente l'app e importa l'allenamento.
   * Se l'app non è installata -> Mostra una landing page con istruzioni.

== 🔒 Privacy Policy

Questa applicazione non raccoglie dati personali, non traccia la posizione e non invia dati a server esterni.
